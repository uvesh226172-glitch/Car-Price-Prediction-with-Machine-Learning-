# Car-Price-Prediction-with-Machine-Learning
# 🚗 Car Price Prediction App

This project is a **Machine Learning-based Car Price Predictor** built using **Python, Scikit-learn, and Streamlit**.  
It predicts the selling price of a car based on features like present price, kilometers driven, fuel type, transmission, and more.

---

## 📌 Features

- ✅ Predict car prices using **Linear Regression**
- ✅ Feature engineering with **Car Age**
- ✅ Encoded categorical variables for model training
- ✅ Interactive web app using **Streamlit**
- ✅ Simple and user-friendly UI

---

## 📂 Dataset

The dataset used contains the following features:

- Car Name  
- Year  
- Present Price  
- Kms Driven  
- Fuel Type  
- Selling Type  
- Transmission  
- Owner  
- Selling Price (Target)

---

## ⚙️ Technologies Used

- **Python**
- **Pandas**
- **NumPy**
- **Scikit-learn**
- **Streamlit**
- **Matplotlib**

---

## 🧠 Machine Learning Workflow

1. **Data Preprocessing**
   - Removed unnecessary columns
   - Converted `Year` → `Car Age`

2. **Feature Engineering**
   - Created `Car_Age` feature

3. **Encoding**
   - Converted categorical data using Label Encoding

4. **Model Training**
   - Used **Linear Regression**

5. **Prediction**
   - Model predicts car price based on user input
